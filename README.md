# webpack5.74_demo

##技术栈
> webpack5、vue3、vue-router、pinia、element-plus、node、es6、typescript、sass、eslint、prettier
其中 pinia、node、typescript正在添加中

## 依赖安装
```
npm i
```

## 启动前端服务
```
npm run start
```

## 启动后端服务
```
npm run server
```

## 构建开发环境包
```
npm run build:dev
```

## 构建线上环境包
```
npm run build
```

## 分析构建的包
```
npm run generateAnalyzerFile
npm run analyzer
```

## 分析构建的包
```

```

## jslint
```
npm run lint
npm run lint-fix
```
